FAQ

But what exactly SWC does?

It cleans temporary files by deleting all files in the three temporary files windows folder. The first one is the temp folder located in "C:\Windows\Temp", the second temp folder located in "C:\Users\"YourUsername"\AppData\Local\Temp", and the third folder is the prefetch folder located in "C:\Windows\Prefetch".

Why delete temporary files?

Acording with the University of Colorado "Too many temporary files can consume valuable storage space. These files accumulate over time and impact the memory available for your computer to run core processes. If youâ€™re running low on storage space, you should consider deleting your temp files to free up space that you can use for other files and data." for more information, checkout the page who talks about this.

It's safe delete all this files?

Yes it is, as said before, all files in these folders are unecessary files led by programs you use or install. If you still in doubt, check out a question about it in Microsoft's official website: Answers.microsoft.

What else SWC can do?

Aside of cleaning the temporary files, SWC can aslo run a disk cleanup to clean your local disk and free some storage space. This is all SWC can do in currently version v.1.1.0 , but in the future some new features will become avaliable.
